# 블로그 작성 전략

## 1. 어떤 식으로 작성하면 좋은가

### 1.1 성분 중심 구조 (1 성분 = 1 글 + 1 계산기)

- **한 글 = 한 성분**에 집중한다. (예: 마그네슘, 비타민 D, 오메가3)
- 각 성분 글 안에 **맞춤 계산기** 링크를 넣어, “내게 맞는 양·형태”를 바로 확인하게 한다.
- 제목·소제목에 **검색어**(성분명, 증상, “권장량”, “부작용” 등)를 자연스럽게 넣는다.

### 1.2 권장 글 구조 (성분별 공통 템플릿)

```
1. 한 줄 요약 (이 성분이 뭐고, 왜 중요한지)
2. 역할·효능 (간단 리스트 + 참고 문헌 링크)
3. 권장량·상한 (성별·연령, 단위 명시) → 여기서 계산기 링크
4. 부족 시·과다 시 증상 (짧게)
5. 형태·제품 선택 팁 (예: 마그네슘 글리시네이트 vs 시트레이트)
6. 주의사항·상호작용 (약, 다른 영양소)
7. 참고 문헌·면책 (NIH, 한국영양학회 등)
8. [내 맞춤 계산기] CTA 버튼/링크
```

- 길이는 **스크롤 부담 없이** 1,500~3,000자 내외를 기본으로, 필요 시 시리즈로 나눈다.

### 1.3 톤·스타일

- **반말/존댓말** 중 하나로 통일 (예: 존댓말 + “~해 보세요”, “~에 도움이 될 수 있어요”).
- “효능” 표현 시 **과장 금지**: “도움이 될 수 있다”, “일부 연구에서”, “권장량을 참고하여” 등으로 완곡 표현.
- 숫자(권장량, 상한)는 **출처 명시** (예: NIH ODS, 2020 한국인 영양소 섭취기준).

## 2. 다른 건강 블로그와의 차별화

| 구분 | 일반 건강 블로그 | 본 블로그 (제안) |
|------|------------------|-------------------|
| **정보** | 일반론 위주 | **성분 단위** + 권장량·형태까지 구체화 |
| **신뢰** | 출처 없음 많음 | **참고 문헌·면책** 필수, 계산식 공개 |
| **활용** | 읽고 끝 | **맞춤 계산기**로 “내 용량·형태” 바로 확인 |
| **표현** | “먹으면 좋다” 위주 | “권장량·상한·주의”까지 써서 **행동 가이드** 제공 |
| **정리** | 긴 글만 | **인포그래픽/한눈에 보기** 페이지로 요약 제공 |

**차별화 문장 (블로그 소개용)**  
“성분별로 **얼마나, 어떤 형태로** 먹으면 좋을지 **계산기로 한 번에** 확인하고, 근거가 있는 정보만 정리한 블로그입니다.”

## 3. 성분 글 작성 시 체크리스트

- [ ] 제목에 성분명 + 검색 의도 1개 (권장량 / 부작용 / 효능 등)
- [ ] 권장량·상한 숫자 + 단위(mg, μg, IU 등) 명시
- [ ] 해당 성분 **맞춤 계산기** 링크 1곳 이상 (상단·권장량 문단)
- [ ] 참고 문헌 1개 이상 + “참고용이며, 진단·치료 대체 불가” 면책
- [ ] 관련 제품 링크 시 제휴/광고 표시 (정책에 따라)

이 구조를 기본으로 하면, 성분 추가 시에도 **일관된 품질**과 **계산기 연동**을 유지하기 쉽습니다.
