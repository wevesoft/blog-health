# 샘플 블로그 가이드

## 샘플 위치

- **전체 예시 글**: [blog/magnesium.md](./blog/magnesium.md) (마그네슘)

## 샘플에서 쓰인 구조

1. **제목**: 성분명 + “완벽 가이드” + 권장량·형태·주의사항 (검색어 포함)
2. **한 줄 요약**: 성분이 뭐고, 왜 중요한지
3. **역할·효능**: 불릿 리스트 + “도움이 될 수 있다” 수준 표현
4. **권장량·상한**: NIH RDA 수치 + **계산기 링크 2곳** (본문·하단 CTA)
5. **부족·과다**: 짧은 증상 정리
6. **형태 선택**: 표로 비교 + 계산기에서 추천 형태 안내
7. **주의·상호작용**: 약물·신장·다른 미네랄
8. **참고 문헌·면책**: 링크 + 진단 대체 불가 문구
9. **CTA**: “맞춤 계산기로 이동” 버튼/링크

## 새 성분 글 쓸 때

- `blog/magnesium.md` 를 복사한 뒤, 성분명·수치·형태·참고 문헌만 바꾼다.
- 계산기 링크는 `ingredient-calc-map.md` 에 등록된 경로로 넣는다.
- 제휴/광고 문구는 사이트 정책에 맞춰 하단에 추가한다.
