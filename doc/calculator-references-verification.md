# 계산기 참고 문헌 검증 결과

각 계산기가 **참고 문헌을 실제로 확인한 뒤** 만들어졌는지, 어디까지 문헌과 일치하고 어디는 단순화·해석인지 정리했습니다.

---

## 1. 마그네슘 계산기

| 항목 | 검증 여부 | 비고 |
|------|-----------|------|
| 기본 RDA (남 400, 여 310) | ✅ NIH ODS 확인 후 반영 | NIH Table 1: 19-30세 남 400 mg, 여 310 mg; 31-50세 남 420, 여 320 |
| 상한 450 mg | ✅ 참고 | NIH UL 등 참고하여 보충 시 권장 상한으로 적용 |
| 운동·카페인·증상 보정 | ⚠️ 단순화 모델 | 운동 시 요구량 증가는 문헌 있음(ACSM 등). 카페인·증상 가산은 참고용 |
| 참고 문헌 링크 | ✅ 실제 문서 링크 | NIH ODS Magnesium, 한국영양학회 |

**결론**: 기본 권장량은 NIH ODS 기준으로 확인·반영됨. 보정값은 단순화된 참고 모델.

---

## 2. 비타민 D 계산기

| 항목 | 검증 여부 | 비고 |
|------|-----------|------|
| 기본 RDA (19-70세 600 IU, 71+ 800 IU) | ✅ NIH ODS 확인 | NIH Table 2: 19-50·51-70세 15 mcg(600 IU), >70세 20 mcg(800 IU) |
| 상한 4,000 IU | ✅ NIH ODS | UL 4,000 IU 명시 |
| 일조량 보정 (+200, -100 등) | ⚠️ 단순화 모델 | NIH RDA는 “최소 일조 가정”으로 정해짐. 일조 많음/적음에 따른 ±는 공식 수치 없음 → 참고용 가산 |
| 목적(뼈/면역) 보정 | ⚠️ 단순화 모델 | 문헌에 “목적별 추가 mg” 공식 없음 → 참고용 |
| 비만 시 +200 | ⚠️ 해석 반영 | NIH는 비만·위험군 언급하나 구체적 “+N IU” 없음 → 참고용 가산 |
| 참고 문헌 링크 | ✅ 실제 문서 링크 | NIH ODS Vitamin D, 한국영양학회 |

**결론**: 기본 RDA·상한은 NIH ODS와 일치. 일조량·목적·비만 보정은 문헌에 명시된 공식이 아닌 **단순화된 참고 모델**.

---

## 3. 철분 계산기

| 항목 | 검증 여부 | 비고 |
|------|-----------|------|
| 남 8 mg, 여 19-50세 18 mg, 51+ 8 mg | ✅ NIH ODS 확인 | NIH Table 1과 일치 |
| 임신 27 mg | ✅ NIH ODS | Table 1 Pregnancy 27 mg |
| 수유 | ✅ 수정함 | NIH는 수유 9 mg. 계산기에서 10 mg → 9 mg으로 수정 |
| 피로/빈혈 보정 (+5, +10) | ⚠️ 참고용 가산 | RDA에 “증상별 추가 mg” 공식 없음. 빈혈은 진단·처방 필요 |
| 상한 45 mg | ✅ 참고 | NIH: 보충제 45 mg/일 이상 시 위장 부작용 가능 |
| 참고 문헌 링크 | ✅ 실제 문서 링크 | NIH ODS Iron, 한국영양학회 |

**결론**: 기본 RDA(성별·연령·임신·수유)는 NIH ODS와 일치. 수유 9 mg으로 반영함. 증상 보정은 **참고용 가산**이며, 빈혈은 의료진 진단 필요.

---

## 4. 오메가3 (EPA/DHA) 계산기

| 항목 | 검증 여부 | 비고 |
|------|-----------|------|
| 일반 500 mg, 심혈관 1,000 mg | ✅ 가이드라인 참고 | AHA 등: 일반 성인 500 mg, 심혈관 질환 850~1,000 mg |
| 생선 섭취·체중 보정 | ⚠️ 단순화 모델 | 공식 “±N mg” 수식 없음 → 참고용 |
| 참고 문헌 링크 | ✅ AHA 링크 | 계산기 내 AHA Science Advisory 링크 있음. 제작 시 해당 문서 직접 확인은 하지 않음 |

**결론**: 기본 500/1,000 mg은 AHA 등 권장과 방향 일치. 생선·체중 보정은 **단순화된 참고 모델**. 필요 시 AHA 문서 직접 확인 후 문구 보강 권장.

| 항목 | 검증 여부 | 비고 |

---

## 요약

- **문헌 확인 후 반영된 부분**: 마그네슘·비타민 D·철분·칼슘·아연·비타민 C의 **기본 RDA·상한**은 NIH ODS(및 계산기 내 링크)와 맞춰져 있음. 철분 수유는 NIH 9 mg으로 수정함. 비타민 C 흡연·임신·수유는 NIH 반영.
- **단순화·참고용인 부분**: 일조량·목적·비만(비타민 D), 증상(철분), 생선·체중(오메가3), 유제품(칼슘), 목적(아연) 등 **보정값**은 공식 RDA가 아니라 참고용 모델로 두었음. 계산기 화면의 “참고 문헌” 블록에서 “보정은 단순화된 모델”이라고 안내하는 것이 좋음.

이 문서는 계산기 참고 문헌 검증 결과를 남겨 두기 위한 것입니다. 추후 문헌을 다시 확인하거나 수치를 수정할 때 이 파일을 갱신하면 됩니다.
