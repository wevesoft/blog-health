# 워드프레스 인포그래픽 템플릿

## 사용 방법

1. **복사할 파일**: `wp-infographic-magnesium.html`
2. **복사 범위**: 파일에서 **`<style>`** 부터 **`</script>`** 까지 통째로 선택해 복사합니다.  
   (맨 위 HTML 주석은 복사해도 되고, `<!DOCTYPE>` / `<html>` / `<head>` / `<body>` 는 없으므로 그대로 붙여넣으면 됩니다.)
3. **워드프레스**: 글/페이지 편집 → 블록 추가 → **사용자 정의 HTML** 선택 → 복사한 내용 붙여넣기.
4. **계산기 링크**: `data-calc-url="https://YOUR-SITE.com/..."` 값을 실제 계산기 페이지 주소로 수정합니다.

## 템플릿 구조 (다른 주제 만들 때 참고)

- **H2**: 섹션 제목 (예: 마그네슘의 역할과 효능, 일일 권장량, 형태별 선택 가이드)
- **H3**: 하위 제목 (예: 부족 시, 과다 시)
- **H4**: 카드/소제목 (예: 에너지 대사, 글리시네이트)
- **클래스**: `.mg-shell`(카드 컨테이너), `.mg-section`, `.mg-header`, `.mg-role`, `.mg-type-card` 등 그대로 두고 내용만 바꾸면 됩니다.

## 글자 크기 (모바일·PC)

- PC: 본문 약 17px, H2 1.22rem, H3 1.08rem, H4 1rem
- 모바일(520px 이하): 본문 16px, H2 1.15rem, H3 1.05rem, H4 0.98rem, 카드 화면 가로 꽉 차게 적용

다른 영양소(철분, 비타민 등)도 같은 HTML/CSS 구조로 복사한 뒤 텍스트와 `data-calc-url`만 바꿔서 사용하면 됩니다.
