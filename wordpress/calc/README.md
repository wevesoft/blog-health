# 워드프레스 계산기 스니펫

## 공통 포맷 (모든 계산기 공통 규칙)

- **복사 범위**: 각 파일에서 `<style>` 부터 `</script>` 까지 통째로 복사 후, 워드프레스 **사용자 정의 HTML** 블록 또는 페이지에 붙여넣기.
- **레이아웃**: 다크 배경, 중앙 카드 하나 (`*.calc-wrap`), `H2 → H3 → H4` 계층 사용.
- **SEO**:
  - H2: `나의 정보 입력`, `계산 결과`
  - H3: 결과 카드 제목(일일 권장량, 추천 형태, 섭취 방법, 참고 팁 등)
  - H4: 참고 문헌 섹션 제목
- **모바일**: 520px 이하에서 카드 폭 100%, 글자 크기 17px 기준.
- **폼 요소**: `min-height: 50~52px`, `padding-right` 넉넉히 확보해 셀렉트 텍스트 잘리지 않도록 통일.

## 마그네슘 계산기 (wp-magnesium-calc.html)

- **설명**: 성별·체중·운동·카페인·증상을 반영한 **마그네슘 권장량 계산기**.
- **포맷**: 이 파일의 구조와 스타일을 **모든 새로운 계산기의 기준 템플릿**으로 사용합니다.

## 비타민 D 계산기 (wp-vitamin-d-calc.html)

- **설명**: 연령·일조량·목적·임신·비만 여부를 반영한 **비타민 D 권장량 계산기**.
- **포맷**: `wp-magnesium-calc.html` 과 동일한 다크 테마, 카드 레이아웃, H2/H3/H4 계층을 사용합니다.
- **로직**: NIH 비타민 D RDA(600~800 IU, UL 4,000 IU)를 기본으로 일조량·목적·비만에 따라 단순 가산/감산.
