# HTML 인포그래픽 — 사용 안내

## 파일

| 파일 | 용도 |
|------|------|
| `magnesium.html` | 마그네슘 가이드 전체 페이지 (독립 실행·미리보기용) |
| `magnesium-wordpress.html` | 워드프레스에 붙여넣을 **본문 조각** (상단 설명 포함) |

## 워드프레스에 적용하는 방법

1. **계산기 링크 주소 확인**  
   실제 사이트에서 계산기 페이지 URL을 정합니다.  
   예: `https://내도메인.com/magnesume_calc.html` 또는 `https://내도메인.com/calculator/magnesium/`

2. **블록에 붙여넣기**  
   - 글/페이지 편집에서 **「+」 블록 추가 → 「사용자 정의 HTML」** 선택  
   - `magnesium-wordpress.html` 파일 내용 **전체**를 복사해 붙여넣습니다.

3. **URL 치환**  
   붙여넣은 코드 안에서 `../magnesume_calc.html` 를 **실제 계산기 주소**로 바꿉니다.  
   (예: `https://내도메인.com/magnesume_calc.html`)  
   `magnesium-wordpress.html` 상단의 `CALC_URL` 부분을 수정한 뒤 사용해도 됩니다.

4. **미리보기**  
   저장 후 미리보기로 레이아웃·색상이 테마와 잘 맞는지 확인합니다.  
   테마 CSS와 충돌하면, 테마 설정이나 「추가 CSS」에서 `.mg-guide` 로 시작하는 클래스만 조정해 보세요.

## 포함된 요소

- **컬러 카드**: 역할·권장량·부족·과다·형태·주의·참고 문헌
- **권장량 바 그래프**: 여성 310 / 남성 420 / 상한 450 (시각 막대)
- **형태별 카드**: 글리시네이트·시트레이트·말레이트·산화물 (컬러 상단 바)
- **체험 예시**: 「나에게 맞는 형태는?」 퀵 선택 → 추천 형태 + 계산기 링크

이미지는 아이콘(이모지)·CSS 그라데이션으로 처리해 별도 이미지 파일 없이 동작합니다.
